
🎯 项目规划：SSML 语音合成平台

1. 核心功能

• ✅ 网页界面输入 SSML
• ✅ 配置 Azure API 密钥和区域
• ✅ 实时语音合成
• ✅ 在线播放 + 下载
• ✅ REST API 接口

2. 技术架构

前端 (HTML/CSS/JS)
  ↓ HTTP
后端 (Python FastAPI)
  ↓ REST API
Azure TTS 服务
  ↓ 返回音频
前端播放/下载3. 详细规划

A. 前端界面 (index.html)

1. SSML 编辑器 (带语法高亮) 可以参考 https://github.com/mekumiao/ssml-editor
2. API 配置表单 (密钥、区域、语音)
3. 控制面板 (合成、播放、下载)

5. 历史记录 (保存的 SSML 模板)B. 后端 API (FastAPI)

端点1：语音合成

POST /api/synthesize
Content-Type: application/json

{
  "ssml": "<speak>...</speak>",
  "api_key": "your_key",
  "region": "eastus",
  "voice": "zh-CN-XiaoxiaoNeural",
  "output_format": "audio-16khz-32kbitrate-mono-mp3"
}

响应：
{
  "success": true,
  "audio_url": "/api/audio/abc123.mp3",
  "duration_ms": 1500,
  "file_size": 24576
}端点2：获取语音列表

GET /api/voices?region=eastus&locale=zh-CN

响应：
{
  "voices": [
    {
      "name": "zh-CN-XiaoxiaoNeural",
      "locale": "zh-CN",
      "gender": "Female",
      "style_list": ["general", "assistant", "chat"]
    }
  ]
}端点3：SSML 验证

POST /api/validate
Content-Type: application/json

{
  "ssml": "<speak>测试</speak>"
}

响应：
{
  "valid": true,
  "errors": [],
  "warnings": ["建议添加语言属性"]
}6. 前端功能细节
